Guessing The Word

You will have 5 chances to guess a five-letter word within 1-5 minutes. You're given a one letter hint. After submitting each guess, your answer will be displayed with each letter color coded as follows: GRAY: The letter is not in the word. GREEN: The letter is in the word, wrong position. BLUE: The letter is in the word, correct position. When you guess the correct word, all letters will be displayed in BLUE. Set Timer for 1-5 minutes (default 3), then press START to play. You'll be able to repeat playing dbWord multiple times.
